Welcome to Dusk.Tools!

Introduction

Dusk.Tools is a code generator for TSWOW that builds code in typescript from a graphical interface. The user may export their code by copying the code or downloading a JSON that can be imported into the tool by another user.

Getting started

Click on any of the three links in the header (Quest, Item or NPC) to navigate to their respective code generators. Fill the "title" field to get the code generating. Code is generated on the fly as you fill out more fields.

Exporting

As mentioned above, users can export their work to either use it in game or pass it to others.

Using it in game

Look for the icon and click it to copy the code to clipboard.

Follow the Module creation guide from TSWOW to create a seperate module, this module will house your additions to the game.

Create a datascript as shown in the guide, paste in your code and save it (CTRL+S)

Save the JSON locally and share it with others.
